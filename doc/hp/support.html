Support

Currently we do not provide a seperate mailing list for the Mobility Framework. However there is a general mailing list available for OMNeT++ which can also be used for MF related issues.

To subscribe, please send an e-mail message containing the line
subscribe omnetpp-l
in the message body to the address omnetpp-l-request@omnetpp.org.

To unsubscribe, send an e-mail with the line
unsubscribe omnetpp-l
in the message body to the address omnetpp-l-request@omnetpp.org.

To post a message, send mail to omnetpp-l@omnetpp.org.
